# toyshop-react
A demo app to implement React core functionalities.

### LIVE DEMO
To see the live app, [GO HERE](https://suvro404.github.io/toyshop-live).

## Project setup
```
npm install
```

### Run the app in the development mode.
```
npm start
```

### Builds the app for production to the `build` folder.
```
npm run build
```
